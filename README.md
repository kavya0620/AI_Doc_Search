# AI DOC Search
AI DOC Search is a Python-based application that allows users to search and query documents using AI techniques

## Features
-upload and process documents
-semantic search using embeddings
-Fast retrieval using Chroma DB
-Simple and interactive interface

##Tech Stack
-python
-Streamlit
-ChromaDB
-LangChain
